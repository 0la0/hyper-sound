# Hyper-Sound
Audio synthesis with HTML.

Development ongoing, documentation to follow.

[Examples](examples.md)

## Build scripts
install dependencies: `npm install`  
run tests: `npm test`  
run linter: `npm run lint`  
build: `npm run build`
